최근 업데이트 사항

04-06 업데이트

1. 내일 국내 현황 예측 기능 추가
- 최근 데이터를 기반으로 계산되는 단순 추정치로 참고용으로만 사용해주세요.
- 전세계 감염자 예측 기능은 삭제하였습니다.
2. 현재 격리 중 환자 수(국내) 추가(현황 탭)
3. 검사 대비 확진율 추가(현황 탭 > 검사 현황 표)

04-04 업데이트

1. 격리해제자, 사망자도 전일 대비 증가치 표시
2. 전세계 감염자 수 예측 정보 추가(현황 탭, 테스트 기능)
- 최근 확진자 추이를 분석해 실시간 감염자 수를 예측합니다.
3. UI 개선(Card 곡률 조정)
4. 성능 안정화(로딩 속도 최적화)

* 일일 현황 업데이트는 업데이트 내역에 포함되지 않습니다.
